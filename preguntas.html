Preguntas de Git y GitHub

1-¿Cómo se llama el lugar donde se almacena todo el contenido y el historial de un proyecto en Git o GitHub, ya sea de forma local o en la nube?
2-¿Cuál es el nombre que suele tener por defecto la rama principal de un repositorio en Git, donde se guarda la versión más estable del proyecto?
3-¿Qué nombre recibe el proceso de combinar los cambios de una rama con otra dentro de un repositorio Git?
4-¿Qué acción se realiza en GitHub para crear una copia de un repositorio en tu propia cuenta y modificarla sin afectar el proyecto original?
5-¿Qué comando se usa para copiar un repositorio remoto desde GitHub a tu computadora y así trabajar localmente con él?
6-¿Cómo se llama la solicitud que se hace en GitHub para que los cambios hechos en una rama se revisen y, si están correctos, se integren en otra rama (generalmente la principal)?
